# GSJS

一个VScode插件，用于提高GS7前端开发效率。

目前实现的功能：

- [ ] 列表、卡片、控件、平台框架、资金框架（zj.core.js）的代码片段

  
## 如何使用




## 本地运行

首先下载[源码](https://github.com/xjonson/vscode-extension-gsjs)，安装依赖

```
npm install
```

安装完后，`F5` 即可运行调试

打包

```
vsce package
```
